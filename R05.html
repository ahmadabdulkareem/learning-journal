color

Foreground Color


The color property allows us to specify the color
of text inside an element.
There are three ways to specify colors:
1. rgb (Red, Green and Blue) values : gb(100,100,90)
2. hex Codes : #ee3e80
3. Color names : DarkCyan

background-color


example:


body { background-color: rgb(200,200,200);}
h1 { background-color: DarkCyan;}
h2 { background-color: #ee3e80;}
p { background-color: white;}

Tips:

Color pickers can help you find the color you want.
